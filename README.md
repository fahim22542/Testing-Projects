## Projects Overview

### Web Automation Practice
**File:** [playwright_automation_practice.py](playwright_automation_practice.py)

A comprehensive web automation script for testing. Features include:
- **Dynamic dropdown handling**: Extracts and processes dropdown options from web elements
- **Loading state management**: Waits for loading indicators to disappear
- **Configurable filter limits**: Supports filtering by hierarchical location parameters
- **Web automation**: Demonstrates browser automation for testing purposes
- **Data extraction**: Collects and processes web data for validation

**Key Functions:**
- `wait_for_loading()`: Waits for loading spinners and overlays to disappear
- `get_dropdown_options()`: Retrieves all available options from dropdown menus
